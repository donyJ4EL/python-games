# Aircraft War Game (飞机大战)

A simple and fun aircraft shooting game built with Python and Pygame.

## Description
Asteroids-style shooter game where you control a spaceship to shoot down enemy aircraft.

## Features
- Player-controlled aircraft
- Enemy aircraft that move downward
- Shooting mechanics
- Score tracking
- Simple collision detection

## Requirements
- Python 3.6+
- Pygame library

## Installation
1. Clone this repository
2. Install dependencies: `pip install pygame`
3. Run the game: `python aircraft_war.py`

## Controls
- Arrow Keys: Move the aircraft
- Spacebar: Shoot bullets

## How to Play
- Destroy enemy aircraft to earn points
- Avoid colliding with enemy aircraft
- Try to achieve the highest score possible!
